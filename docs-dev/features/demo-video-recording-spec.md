# Demo 操作视频录制规格（BDD）

## 背景
本期需要产出一个可复用的操作示例视频，用于展示 `react-debug-inspector` 的核心工作流。

## 验收范围
- 视频主题固定为：使用 Debug 工具选择元素后，将复制内容粘贴到输入框。
- 录制画面中必须可见鼠标移动与点击过程。
- 提供可重复执行的一键录制命令，避免手工录屏不一致。

## 场景 1：核心操作链路可完整演示
- **Given** 用户打开 Demo 页面
- **When** 用户点击右下角 `🎯` 进入检查模式
- **And** 用户点击目标元素触发复制
- **And** 用户聚焦输入框并执行粘贴
- **Then** 输入框中应出现包含 `debug id` 的文本

## 场景 2：录制视频中可见鼠标
- **Given** 用户执行演示录制命令
- **When** 脚本驱动鼠标在页面中移动并点击
- **Then** 录制视频中应能清晰看到鼠标轨迹与点击位置

## 场景 3：录制结果可稳定产出
- **Given** 本地依赖安装完成
- **When** 执行录制命令
- **Then** 应产生视频文件（Playwright video）
- **And** 失败时可通过 Playwright 报告定位问题
